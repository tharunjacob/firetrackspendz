# TrackSpendz - Tech Stack Summary

## Frontend Framework
- **React 19** with **TypeScript** - Modern UI library with type safety
- **Vite** - Fast build tool and dev server

## UI & Styling
- **Tailwind CSS** - Utility-first CSS framework (via CDN)
- **Recharts** - React charting library for data visualizations

## Data Storage
- **IndexedDB** - Browser-based local database for client-side data persistence

## AI Integration
- **Google Gemini AI** (`@google/genai`) - For intelligent transaction categorization and financial advice

## File Processing
- **XLSX** (SheetJS) - Excel/CSV file parsing and processing

## PDF Generation
- **jsPDF** - PDF document generation
- **html2canvas** - Convert HTML/DOM to canvas for PDF export

## Deployment
- **Vercel** - Hosting and deployment platform
- **GitHub** - Version control and repository hosting

## Development Tools
- **TypeScript** - Type-safe JavaScript
- **ESBuild** (via Vite) - Fast JavaScript bundler

---

**Architecture:** Single Page Application (SPA) with client-side routing, 100% client-side processing for privacy.
