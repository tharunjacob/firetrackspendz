Frequently Asked Questions

Is my financial data safe?

Absolutely. Free tier users store all transaction data locally inside their browser's IndexedDB. Pro tier users benefit from secure cloud sync utilizing end-to-end encrypted databases built on Supabase with Row-Level Security (RLS) policies.

How are statement files processed?

Uploaded statements are parsed in memory using an isolated secure Edge Function to pull transaction text. Raw statement files are never persisted on the server, ensuring complete data security.
