Frequently Asked Questions

Is my financial data safe?

Yes. Free tier data is stored locally on your device via IndexedDB. Pro tier data is protected in transit with HTTPS and at rest in Supabase using Row-Level Security (RLS).

How are PDF files handled?

PDF files are processed in-memory using a secure Edge Function proxy to extract transactions. Raw statements are never saved on the server.
